集中型ログ管理について学習します。このモジュールでは、Loki のセットアップ、Promtail を使用したログの転送、LogQL を使用したログのクエリ、ログからのメトリクス、よりスマートなラベリングについて扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
